## Electron: 設定ファイル (`userData`)

| OS   | Location                                                |
| ---- | ------------------------------------------------------- |
| Mac  | $HOME/Library/Application Support/<アプリ名>             |
| Win  | %USERPROFILE%\AppData\Roaming\<アプリ名>                 |
| Linux| $HOME/.config/<アプリ名>                                 |
