# jest

- https://jestjs.io/

## アサーション

- JestはJasmineをベースとしてその上に作られているので基本的なAssertなどはJasmineと同じです。

## DOM

- JestはjsdomによるDOMの上で実行される
- nodeでのテストのようにcommand line I/Fでテストを実行することが出来ます。
- KarmaのようなTest Runner不要
